# QtScreenshot

## Description

A very simple screenshot tool for Qt and C++

## Build

```bash
cmake -S . -B build
cmake --build build
```

For a debug build, use:
```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```

### For Mac

cmake -B build -S . -DCMAKE_PREFIX_PATH=/Users/sr/Qt/6.9.1/macos





## Usage

```bash
./build/QtScreenshot
```
